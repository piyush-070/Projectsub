-> Expense Project

#Project Overview

Expense Splitter is a lightweight and user-friendly web application designed to manage shared expenses among groups. It helps users track who paid, split expenses equally, and clearly identify who owes whom.

This application is useful for trips, shared living, or team activities where expense tracking is required.

---

1)Features

*  Create multiple groups
*  Add members to each group
*  Add expenses with description and amount
*  Select who paid for the expense
*  Automatically split expenses equally
*  Real-time balance calculation
*  Settlement summary (who owes whom)
*  Expense list display
*  Clean and intuitive UI

---

2) Tech Stack

* HTML
* CSS
* JavaScript

---

3) How to Run Locally

1. Clone the repository:

   ```bash
   git clone https://github.com/piyush-070/Projectsub.git
   ```

2. Open the project folder

3. Run the application:

   * Open `index.html` in any browser

---

4) Application Logic

* Expenses are split equally among all members
* Each user's balance is calculated based on:

  * How much they paid
  * Their share in expenses
* A settlement algorithm minimizes transactions between users
